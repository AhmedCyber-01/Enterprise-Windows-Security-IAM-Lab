# Enterprise Windows Security Monitoring & Identity Access Management Lab

## About This Project

I built this home lab to learn how Active Directory and Identity & Access Management (IAM) work in a Windows environment. Instead of only learning the concepts, I wanted to create my own domain, manage users and groups, apply permissions, and monitor Windows Security logs in Splunk.

This project helped me understand user management, Group Policy, Windows event logging, and basic security monitoring in a hands-on way.

---

## Skills Demonstrated

- Active Directory installation and configuration
- User and group management
- Organizational Units (OUs)
- Role-Based Access Control (RBAC)
- Group Policy configuration
- Windows Security Event monitoring

  
----

## Environment

| Component | Details |
|----------|---------|
| Domain Controller | Windows Server 2022 |
| Client | Windows 10 |
| Domain | `corp.local` |
| Hypervisor | VirtualBox |

## Tools Used

- VirtualBox

## What I Learned

While building this lab, I learned how Active Directory is organized, how permissions are managed using security groups, how Group Policy works, and how Windows Security events can be monitored in Splunk. I also gained experience troubleshooting common configuration issues while setting up the environment.
